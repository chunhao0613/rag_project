# SRE 技術細粒化筆記索引

這個資料夾把原本 01~25 的階段筆記，拆成技術導向學習路徑。

## 使用方式

1. 先看 [學習地圖](./learning-map.md) 了解 01~25 對應關係。
2. 依技術主題閱讀：每份都用 3W（What / Why / Where-How）寫法。
3. 每份都包含：
- 核心名詞表
- 技術間協作關係
- 手把手範例
- 指令參數解說（用途、何時用、怎麼用）
- 常見故障排查
